# bluehatkeem/homebrew-tap

Homebrew tap for [Freepaste](https://github.com/bluehatkeem/freepaste) — a free, menu-bar clipboard manager for macOS.

## Install

```sh
brew install --cask bluehatkeem/tap/freepaste
```

Or tap first, then install:

```sh
brew tap bluehatkeem/tap
brew install --cask freepaste
```

## Update

```sh
brew update
brew upgrade --cask freepaste
```

## Uninstall

```sh
brew uninstall --cask freepaste          # remove the app
brew uninstall --zap --cask freepaste    # also remove history and preferences
```
